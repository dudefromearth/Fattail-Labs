**description_md:**

Stop the bleeding first. This short course builds the capital-preservation mindset and the weekly habits that keep traders from compounding avoidable damage.

You will not get profit promises. You will get a clear order of operations: process before optimization, risk defined before reward language, and a review loop that builds capacity rather than dependency.

**By the end you will be able to:** state a personal stop-the-bleeding rule; define risk in plain language for one setup; run a one-page weekly process review; and use a simple checklist when emotions pull you off process.
