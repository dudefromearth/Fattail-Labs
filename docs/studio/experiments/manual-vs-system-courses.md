# Experiment: Manual course vs System course

**Goal:** Learn **cost and quality** by producing two complete courses:

| Track | Method |
|---|---|
| **Course A — Manual** | Outline + scripts by hand; HeyGen CLI; place on Labs with rich notes |
| **Course B — System** | Board card → blueprint workspace → approve → scripts/skills → produce |

**Measure:** $/lesson, time, quality (1–5), doctrine, how well companion text works.

---

## Course A — Manual (this track first)

**Title:** Stop the Bleeding Foundations  
**Slug:** `stop-the-bleeding-foundations`  
**Level:** beginner  
**Shape:** Header + 2 modules + 6 lessons + KC + resources notes  

| Module | Lessons |
|---|---|
| 1. Foundation | (1) stop-the-bleeding-mindset · (2) define-risk-before-reward · (3) weekly-process-review |
| 2. Practice | (4) process-checklist · (5) emotional-triggers · (6) one-focus-week |

**Video:** dense ~3 min HeyGen (Dude Primary)  
**Companion:** substantial `body_md` per lesson (checklists, recap)  
**Path:** CLI / `live-send/` — not board factory  

**Already rendered (Wave A):** lessons 1–3 (~$16.50, quality 5).  
**Remaining video:** lessons 4–6 (~$16.50 more at same unit cost).
